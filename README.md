# 简介
本储存库是一个能够帮助道友加速修仙的辅助工具。通过对国漫去头去尾，只保留正片内容，让道友用电影的打开方式打开动漫。

🌟 **主要特点：**

1.去片头片尾，无感下一集，电影级享受；

2.4K高码率无损画质，选用各平台最清晰画质剪辑，凡人、沧元图原始文件大约一集3GB，其余动漫大约一集1GB；

3.无任何广告；

4.动漫有分章节，按章节剪辑，没有章节大约8-10集剪辑一次，片长在2小时左右。
